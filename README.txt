MultiView Perceptual Disparity Model tool - MVPDM tool

This software tool implements the perceuptual model for Super Multiview (SMV) video described in: 
- P. Carballeira, J. Gutiérrez, F. Morán, J. Cabrera, F. Jaureguizar, N. García, “Multiview Perceptual Disparity Model for Super Multiview Video”, IEEE Journal of Selected Topics in Signal Processing, Issue on Measuring Quality of Experience for Advanced Media Technologies and Services, vol. xx, no. x, pp. xxxx-xxxx, xxx 2017. (doi:10.1109/JSTSP.2016.2617302)

Requirements:
- Winwdous x64 system
- Change computer regional settings to use dot (.) as the decimal separator

Usage:
- Open MVPDM_GUI.exe
- Browse camera parameter and depth map image files, arc or linear arrangements (not available for .yuv or video yet)
- Push Create MVPDM
- Push Compute d_perc
- Push Compute MVPDM
- Push Compute prediction (only for alpha = 1.5, beta = -0.6  )
- Push Design Linear SMV for linear arrangemets or Design Arc SMV for arc arrangements
