Requirements:
- Change computer regional settings to use dot (.) as the decimal separator


Usage:

- Open MVPDM_GUI.exe

- Browse camera parameter and depth map image files, arc or linear arrangements (not available for .yuv or video yet)
- Push Create MVPDM
- Push Compute d_perc
- Push Compute MVPDM
- Push Compute prediction (only for alpha = 1.5, beta = -0.6  )
- Push Design Linear SMV for linear arrangemets or Design Arc SMV for arc arrangements
